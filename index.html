🌲 TreeForge

Simple, Lightweight Tree Rendering for the Browser

What is TreeForge?

TreeForge is a lightweight JavaScript library that renders hierarchical tree structures (like file systems) in the browser. It's perfect for displaying folder structures, data hierarchies, or any tree-based data.

📦 Lightweight

No dependencies, small footprint

🎨 Customizable

Icons, colors, indent styles

🔄 Interactive

Expand/collapse, CRUD operations

📊 Flexible Data

Local JSON or API backend

⚡ Quick Start (3 Steps)

Step 1: Add HTML container

<div id="tree-container"></div>

Step 2: Prepare your data

const myData = [
  {
    name: 'src',
    type: 'folder',
    children: [
      { name: 'index.js', type: 'file' },
      { name: 'utils.js', type: 'file' }
    ]
  },
  { name: 'package.json', type: 'file' }
];

Step 3: Render the tree

const tree = new TreeForge({
  containerId: 'tree-container',
  localData: myData
});

That's it! Your tree will be rendered with default styling.

🔧 Customization Options

Basic Options

new TreeForge({
  containerId: 'tree-container',
  localData: myData,
  
  settings: {
    asciiStyle: 'unicode',  // or 'ascii'
    indent: 2,              // spaces per level
    icons: {
      folder: '📁',
      file: '📄',
      open: '📂'
    },
    colors: {
      folder: 'cyan',
      file: 'white'
    }
  }
});

With API Backend

new TreeForge({
  containerId: 'tree-container',
  
  // Load data from your API
  onLoad: async (path) => {
    const response = await fetch(`/api/files?path=${path}`);
    return await response.json();
  },
  
  // Handle file operations
  hooks: {
    onCreateFile: async (path, name) => { /* ... */ },
    onDeleteFile: async (path) => { /* ... */ },
    onRenameFile: async (path, newName) => { /* ... */ }
  }
});
